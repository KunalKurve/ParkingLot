ParkPlus Clone - Parking Lot System (Low-Level Design)
An interview-focused, Low-Level Design (LLD) / Machine Coding implementation of a smart parking management system inspired by ParkPlus. This project is designed to showcase clean code practices, Object-Oriented Design (OOD) principles, and design patterns commonly evaluated in technical rounds at top tech companies.

The codebase prioritizes readability, extensibility, and separation of concerns over production-ready infrastructure, making it an ideal reference for machine coding interviews.
